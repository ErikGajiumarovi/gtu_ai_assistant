# Отчёт о возможностях ИИ-ассистента GTU AI Assistant

## Текущие возможности

ИИ-ассистент GTU AI Assistant представляет собой backend-приложение с frontend-интерфейсом, предоставляющее следующие функции:

### Основная функциональность
- **Аутентификация пользователей**:
  - Регистрация нового пользователя (email, имя, фамилия, пароль)
  - Вход в систему с получением JWT-токена
  - Защищённые эндпоинты requiring authentication
- **Управление чатами**:
  - Создание нового чата с ИИ-ассистентом (отправка сообщения пользователя → получение ответа ИИ)
  - Продолжение существующего чата (отправка сообщения → ответ ИИ)
  - Получение списка чатов текущего пользователя
  - Удаление чата
  - Поддержка потоковых (streaming) ответов для создания ощущения реального времени
- **Многопользовательская поддержка**: Каждый пользователь имеет изолированное пространство для своих чатов

### Техническая архитектура
- **Backend**: Kotlin, Ktor, чистая архитектура, DDD, hexagonal architecture
- **Persistent Storage**: PostgreSQL с Exposed, поддержка PGVector для векторного хранения (инфраструктура подготовлена, но пока не используется в генерации ответов ИИ)
- **Dependency Injection**: Koin
- **Обработка ошибок**: Функциональное Either (Arrow) для явного управления ошибками
- **Frontend**: Kotlin/JS с Compose for Web
- **Деплой**: Docker-контейнеризация (backend, frontend, база данных)

### Возможности ИИ-агента
- **Генерация ответов**: Использует LLM (OpenAI через Koog) для создания ответов на основе истории диалога
- **Контекст диалога**: Агент использует последние 20 сообщений из истории чата для генерации ответов
- **Системный промпт**: Фиксированный промпт, определяющий поведение агента (краткие, релевантные, полезные ответы)
- **Потоковая генерация**: Поддержка потоковой выдачи токенов для улучшения UX

### Ограничения текущей реализации
- **Отсутствие интеграции с базой знаний**: Несмотря на наличие инфраструктуры для векторного поиска и приёма знаний, агент в текущей версии не использует внешние знания при генерации ответов (работает только с историей диалога)
- **Отсутствие инструментов**: Нет интеграции с внешними инструментами (веб-поиск, вычисления и т.д.)
- **Отсутствие долговременной памяти**: Память ограничена текущей сессией чата
- **Отсутствие мультимодальности**: Агент работает только с текстовым вводом/выводом
- **Отсутствие персонализации**: Нет адаптации под конкретного пользователя или курс

## Потенциальные улучшения для ИИ-ассистента учебного заведения

### Академическая поддержка
- **Интеграция с базой знаний**:
  - ДобавитьRetrieval-Augmented Generation (RAG) для использования учебных материалов, конспектов лекций, учебников при генерации ответов
  - Позволить загружать и индексировать курсовые материалы в векторное хранилище
- **Курс-специфичные ассистенты**:
  - Возможность привязки ассистента к конкретному курсу с предварительной загрузкой соответствующих материалов
  - Автоматическое суммарирование лекций и создание учебных пособий
- **Помощь с заданиями**:
  - Пошаговое руководство при решении задач (без выдачи прямых ответов)
  - Проверка понимания через контрольные вопросы
  - Генерация похожих задач для закрепления материала
- **Генерация учебных материалов**:
  - Создание карточек для запоминания (flashcards)
  - Генерация тестовых вопросов и вариантов ответов
  - Составление планов подготовки к экзаменам

### Взаимодействие и сотрудничество
- **Система объявлений и уведомлений**:
  - Чтение и суммаризация институциональных объявлений
  - Напоминания о дедлайнах, экзаменах, мероприятиях
- **Поддержка групповой работы**:
  - Фасилитация групповых дискуссий с участием ИИ-ассистента
  - Генерация протоколов собраний и списков действий
  - Помощь в распределении ролей в групповых проектах
- **Изучение языков**:
  - Тренировочные диалоги на иностранных языках с корректировкой ошибок
  - Перевод и объяснение академических текстов
  - Генерация упражнений по грамматике и лексике

### Административные и организационные функции
- **Интеграция с календарем**:
  - Двусторонняя синхронизация с календарными системами (Google, Outlook)
  - Уведомления о предстоящих событиях и дедлайнах
  - Помощь в планировании времени на учёбу и отдых
- **База часто задаваемых вопросов (FAQ)**:
  - Автоматическое отвечание на типовые вопросы о правилах, процедурах, ресурсах вуза
  - Обучение на основе исторических запросов пользователей
- **Навигатор по ресурсам кампуса**:
  - Информация о часах работы библиотек, лабораторий, центров репетиторства
  - Помощь в поиске свободных аудиторий и оборудования
- **Функции доступности**:
  - Преобразование текста в речь и наоборот
  - Генерация материалов в альтернативных форматах (крупный шрифт, совместимость со скрин-ридерами)

### Технические улучшения
- **Мультимодальный ввод**:
  - Распознавание текста с изображений (OCR) для работы с учебными материалами
  - Аудио-ввод для лекций и голосовых заметок
  - Понимание и описание изображений (диаграммы, графики, формулы)
- **Персонализация и адаптивное обучение**:
  - Отслеживание прогресса пользователя по темам и курсам
  - Рекомендация персонализированных ресурсов и упражнений
  - Выявление пробелов в знаниях и предложение целенаправленной работы
- **Объяснимый ИИ (XAI)**:
  - Указание источников при использовании внешних знаний
  - Показ этапов рассуждения при сложных выводах
  - Возможность проверки фактов и цитирования
- **Этические и безопасностные механизмы**:
  - Встроенные проверки на академическую честность (обнаружение попыток списывания)
  - Снижение алгоритмической предвзятости
  - Соответствие требованиям конфиденциальности данных (FERPA, GDPR для образовательных данных)
  - Контроль над генерацией неподобающего контента

## Заключение

GTU AI Assistant уже обладает прочной основой для разговорного ИИ-агента с аутентификацией и управлением чатами. Интеграция существующей инфраструктуры базы знаний и добавление возможностей, специфичных для образовательной среды, позволит превратить его в комплексного ИИ-ассистента, способного поддерживать учебный процесс, повышать успеваемость студентов и улучшать взаимодействие в академическом сообществе GTU.
